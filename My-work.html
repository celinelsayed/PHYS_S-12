Assignment: Create a website and make a page with a proposal for a possible final project. Include an index.html page with links to each assignment page, an about page, and some customization using HTML and CSS

Having only used python before, this was my first time using HTML, so I've completely relied on templates, youtube tutorials and even just looking at the HTML of different websites I find aesthetically pleasing and taking inspiration from their code (mainly CSS). I ended up devoting alot of time to it, and it became something i enjoyed and instead used as a way to procastinate my other work (especially since I'm still in school). I used websites such as w3 schools and nicepages to browse the greatest variety of webpages I could, and adapt them to my liking and to suit me more.
